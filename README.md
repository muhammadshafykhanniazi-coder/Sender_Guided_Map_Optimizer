# Sender_Guided_Map_Optimizer
Our Sender-Guided Pathfinding Project is an intelligent navigation system designed to find optimal routes on a graph-based map. The user, acting as the sender, specifies the start and goal nodes, and the system dynamically explores possible paths using advanced search algorithms such as A, UCS, or Greedy Best-First Search. It visually demonstrates node traversal, path selection, and decision-making in real time, providing insights into algorithm efficiency and behavior. This project not only serves as a learning tool for understanding pathfinding and graph theory but also simulates real-world applications like delivery routing, robotics navigation, and network data transfer
